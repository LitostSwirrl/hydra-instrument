Mouse & Touch
CANVAS INTERACTION
mouse position → mappable to any param (default: rotate + hue shift)
click + drag → XY pad mode (pitch bend + filter cutoff)
scroll wheel → zoom visual scale
touch (multi) → each finger is an independent XY source
CONTROL PANEL WIDGETS
sliders → continuous params (freq, reverb wet, etc.)
dropdowns → discrete choices (synth type, source function)
toggles → on/off (effects bypass, mic input)
XY pad → 2D control (filter freq + resonance)

Technical Stack Summary

  • React + TypeScript + Vite -- SPA, no backend needed
  • Tone.js -- audio synthesis, effects, transport/sequencer
  • hydra-synth -- visual engine, embedded via npm package
  • Custom GLSL -- 6 shader functions via Hydra's setFunction() API
  • Tailwind CSS -- minimal dark UI, sans-serif throughout
  • localStorage -- preset persistence. JSON export/import for sharing
  • No backend -- entirely client-side. Deploy to GitHub Pages
