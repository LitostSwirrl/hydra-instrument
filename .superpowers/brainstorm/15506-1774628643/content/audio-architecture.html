Key Decisions

  • Tone.js PolySynth as keyboard instrument -- switchable between FM, AM, Membrane, and custom synth types. Polyphonic (play chords)
  • Step sequencer using Tone.Transport -- pattern-based, programmable beats and melodies that loop. BPM-synced
  • Mic/line input via getUserMedia -- can also capture system audio (browser-dependent). Feeds through same effects chain
  • Effects are shared -- all three sources mix into one effects chain. Each effect is toggle-able with wet/dry control
  • Dual analysis -- FFT bands (configurable 4-16) and waveform/envelope follower both feed into Hydra as reactive parameters
  • Keyboard mapping: ASDF row = lower octave, QWERTY row = upper octave. Z/X shift octave down/up
