# UX Audit -- 2026-04-01

User-reported issues and feature requests from first real testing session.

---

## Critical

### 10. Key sustain -- notes cut off after 0.5s
Pressing a key produces sound for ~0.5s then stops, even if held. Non-percussive sounds (sine, triangle, saw) should sustain while held and release on keyup. Current `noteOff` is a no-op and `superdough` uses hardcoded 0.5s duration.

---

## UX Confusion

### 1. Overall UX is confusing
No clear mental model for how the tool works. Users don't understand the relationship between controls, sounds, and visuals.

### 2. Macros vs Sound sliders -- unclear and redundant
Filter/Reverb/Volume sliders appear in both simple and pro mode but it's unclear what they control. No labels explaining they drive both audio effects and visual parameters simultaneously. Users can't tell what the sliders do.

### 3. Onboarding too simplified, missing info
The interactive intro skips important details:
- Upper row keys (W, E, T, Y, U, O, P) trigger sharps/flats (piano black keys) -- never mentioned
- No explanation of what sliders do
- No explanation of the audio-visual mapping concept

### 7. Sound-visual coherence
Default sounds don't match their visual preset aesthetically. When selecting a visual scene, the paired sound should feel like it belongs. Currently feels arbitrary.

### 9. Sound presets are basic
Sine, saw, triangle, square sound like 2000s GarageBand defaults. Need more interesting timbres, layering, or effects to feel like a real instrument.

---

## Feature Requests

### 4. Re-surfaceable help (?) button
Add a `?` SVG glyph (no emoji) in the bottom-right corner that reopens onboarding/help at any time. Currently the help button exists but is low-visibility.

### 5. Start from silence -- black screen, no sound
Currently presets auto-play a sequencer pattern on load. Users want a blank canvas: black screen, silence, build from scratch. The sequencer pattern should be opt-in, not default.

### 6. Poetic preset names
Replace technical names (void, ritual, signal, ember, cosmos, mask) with more evocative names. Reference: Sylvia Plath's frequently used words as inspiration.

### 8. More instrument variety in simple mode
Simple mode only offers synth waveforms (sine, tri, square, saw). Add other instrument types -- piano, strings, pads, bells, etc. -- using Strudel's sample banks.

### 11. Full help/reference panel
The `?` button should open more than just onboarding steps. Include:
- Full keyboard map (which keys play which notes, including sharps)
- What each slider does (audio effect + visual mapping)
- How to start making art (suggested workflow)
- Keyboard shortcuts reference

### 12. Record / Revert / Export toolbar
Add a top-of-page toolbar with:
- **Record**: capture the audiovisual performance
- **Revert**: undo to previous state
- **Export**: save the work (video? audio? preset?) so users can take it home
